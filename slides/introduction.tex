\begin{frame}[fragile]{Hello, world!}


Create a file called {\tt HelloWorld.java} and put the following in it:
\begin{lstlisting}[language=Java]
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, world!");
    }
}
\end{lstlisting}
Now navigate on the command line to the directory containing {\tt HelloWorld.java} and do this (\$ is your command prompt):

\begin{lstlisting}[language=Java]
$ javac HelloWorld.java
$ java HelloWorld
Hello, world!
$ 
\end{lstlisting}

You've just written, compiled, and run your first Java program!

\end{frame}
